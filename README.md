# README

### Ruby version
2.5.1

### System dependencies
postgresql

### Configuration
be sure to add `env.yml` locally and add in credentials for:
```
ZENSEND_KEY: API_KEY
ZENSEND_ORIGINATOR: YOUR_MOBILE 
```

### Database creation
`rake db:setup`


### How to run the test suite
`rspec`
